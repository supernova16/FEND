# Memory Game

It is a memory game (matching game) completed by myself.

## Stack

- HTML,CSS,JavaScript
- JQuery
- [Animate.css](Animate.css)
- [Font Awesome](https://fontawesome.com/)

## How to play

- Click each card
- Find the same cards
- When all the cards match, you win
- You can restart at any time
