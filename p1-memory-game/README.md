# Memory Game

It is a memory game (matching game) completed by myself.

## stack

- HTML,CSS,JavaScript
- JQuery
- [Animate.css](Animate.css)
- [Font Awesome](https://fontawesome.com/)

## how to play

- click the card
- find the same cards
- When all the cards match, you win
- you can restart at any times
